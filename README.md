# ARPAPED - Modular Web3 Super App

یک سوپر اپ ماژولار مبتنی بر Web3 و بلاکچین

## ساختار پروژه

```
src/
├── app/              # Next.js App Router
├── modules/          # ماژول‌های کسب‌وکار
├── shared/           # کامپوننت‌های مشترک
└── core/             # هسته سیستم
```

## نصب و اجرا

```bash
npm install
npm run dev
```

پروژه در http://localhost:3000 اجرا می‌شود

## مستندات

- `KNOWLEDGE_TRANSFER.md` - دانش کامل پروژه
- `ARCHITECTURE.md` - قوانین معماری
- `PROJECT_SETUP.md` - راهنمای راه‌اندازی
- `GIT_COMMANDS.md` - دستورات Git

## تکنولوژی‌ها

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Zustand
